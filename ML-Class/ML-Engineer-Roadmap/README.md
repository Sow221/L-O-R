# ML Engineer Roadmap 🚀

## Ton objectif
Devenir Machine Learning Engineer junior solide en 3-6 mois.

## Setup
1. `cd ML-Engineer-Roadmap`
2. `conda env create -f environment.yml` (or `pip install -r requirements.txt` if no conda)
3. `conda activate ml-roadmap`
4. `jupyter lab` - ouvre browser, navigue vers Phase-1-Foundations/lesson.ipynb

## Structure
- **Phase-1-Foundations/**: Python data basics (Pandas, NumPy...)
- **Phase-2-ML-Basics/**: Scikit-learn intro
- **Phase-3-SmartWasteProject/**: Projet réel déchets
- etc.

## Workflow par phase
1. Lis/execute lesson.ipynb cellule par cellule.
2. Fais exercices dans practice_exercises.py.
3. Documente tes résultats/outputs dans le notebook.
4. ✅ TODO.md et dis-moi pour next phase.

## environment.yml
Python 3.10 + libs essentielles.
